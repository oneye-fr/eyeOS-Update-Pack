eyeOS-Installation
------------------

Um eyeOS zu installieren gehen Sie folgendermaßen vor:

 1. Downloaden Sie die neueste Version von eyeOS von www.eyeos.org.
 2. Dekomprimieren Sie die Datei.
 3. Laden Sie die Dateien und Ordner auf Ihren Webspace.
 4. Ändern Sie CHMOD von dem Ordner "etc" auf 777.
 5. Surfen Sie mit einem Browser zu der Adresse und führen Sie den Installationsschritt aus (nur 1 Schritt).

Die Installation ist fertig, Sie können eyeOS einsetzen.

Bei Problemen:
 - Es erscheint nur ein weißer Bildschirm:
    - Stellen Sie Ihren Webspace auf PHP 4 oder am besten gleich auf PHP 5 um.
 - Fehlermeldungen, melden, dass eine Datei / ein Ordner nicht beschrieben werden kann:
    - Legen Sie eine php.ini an, die dem Inhalt "Safe_mode = Off".
    - Überprüfen Sie den Punkt 4.
