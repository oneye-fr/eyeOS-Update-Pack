Update-Pack
-----------

Um das Update-Pack zu installieren gehen Sie folgendermaßen vor:

 1. Installieren (siehe Installation) Sie eyeOS.
     (Ein Update eines bestehenden Systems ist auch möglich, dann muss dieser Schritt jedoch ausgelassen werden!)
 2. Downloaden Sie die neueste Version vom Upadte-Pack von www.eyeos.de.md.
 3. Dekomprimieren Sie die Datei.
 4. Kopieren Sie nun die Dateien des Update-Packs in Ihr eyeOS-System.
    Ersetzen Sie dabei alle vorhandenen Dateien und Ordner.

Das Update-Pack ist fertg installiert, Sie können es nun verwenden!
