eyeOS-Update
------------

Um eyeOS zu updaten gehen Sie folgendermaßen vor:

 1. Downloaden Sie die neueste Version von eyeOS von www.eyeos.org.
 2. Dekomprimieren Sie die Datei.
 3. Löschen Sie im dekomprimierten Verzeichnis die Ordner "etc", "home" und "usr".
 4. Laden Sie alle übrigen Dateien und Ordner auf Ihren Webspace hoch.
    Ersetzen Sie dabei alle vorhandenen Dateien und Ordner.

Das Update ist fertig, Sie können eyeOS wieder einsetzen.
