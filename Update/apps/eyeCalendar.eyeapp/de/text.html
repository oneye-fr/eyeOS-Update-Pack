Einleitung
EyeCalendar verwaltet Ihre Termine. Es ist der Terminkalender von eyeOS. Nach dem Start ist bereits der heutige Tag aktiviert.

Termine erstellen
Um ein Termin festzulegen wählen Sie rechts den gewünschten Tag und geben Sie dann die Informationen zu dem Tag in die Textbox (links). Klicken Sie dann auf den Speichern-Button. Wählen Sie diesen Tag nun wieder an, so erscheint links der soeben eingetragene Text. Um die Nachricht wieder zu löschen klicken Sie einfach auf den Löschen-Button.

Einstellungen festlegen
Klicken Sie auf das kleine Werkzeug am oberen Rand des Fensters, um die Einstellungen zu öffnen. Hier können Sie festlegen:
Wochennummer anzeigen - Zeigt die Wochennummer links vom Kalender an
Löschen bestätigen - Wenn Sie einen Termin löschen müssen Sie das Löschen bestätigen
Auto-Sichern beim Beenden - Speichert die Termine automatisch
Uhr-Format - Legt das Uhrzeit-Format fest
Symbolleisten-Text - Legt den Symbolleisten-Text fest
Titel-Anzeige - Legt den Titel fest, der über jedem Termin angezeigt wird
Tage, für die beim Start gesucht wird - Legt eine Standardzahl fest, wieviele Tage beim Autostart geprüft werden sollen.
