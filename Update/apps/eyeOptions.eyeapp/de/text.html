Einleitung
EyeOptions ist die Systemkonfiguration für Ihr eyeOS-System.
Der root-Benutzer hat besondere Rechte. Diese finden Sie unter root-Einstellungen.

Desktop ändern
Unter Aussehen und Gestaltung können Sie Themes aktivieren oder den Hintergrund ändern.
Um den Hintergrund zu ändern geben Sie hinter Neuer Hintergrund den Pfad zu Ihrem neuen Hintergrund an und klicken auf Hintergrund ändern.
Über eyeOS-Hintergrund wiederherstellen aktivieren Sie wieder den Standard-Hintergrund.
Das Theme ändern Sie indem Sie es auswählen und dann auf Theme ändern klicken.
Ein Theme beinhaltet einen neuen Hintergrund, sowie neue Icons für Programme und Fenster usw.

Passwort ändern
Wenn Sie Ihr Passwort ändern möchten, geben Sie das alte Passwort und danach zweimal das neue Passowrt ein. Klicken Sie danach auf Passwort ändern.

root-Einstellungen
Der root-Benutzer hat zusätzliche Einstellmöglichkeiten:
Passwort ändern - Ändern Sie das Passwort eines Benutzers
Neuen Systemnamen - Geben Sie einen neuen Systemnamen an
Neuen Benutzer - Legen einen neuen neuen Benutzer an
Benutzer löschen - Löschen Sie einen Benutzer
