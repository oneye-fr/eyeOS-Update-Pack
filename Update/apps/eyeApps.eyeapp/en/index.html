Introduction

This is your applications manager for the eyeOS system. You can access eyeApps by clicking on the Application Dock (or toolbar) icon with the plus sign (+) in it.
The eyeApps window will display all of the applications installed on the system that are available to you. To run any of these applications, simply click on the icon for it.
From this window, you can manage the application settings specifically for your desktop environment, as well as install new applications (if the system's administrator has allowed you access to this feature).

Managing Apps

Click on Manage Apps to bring up the settings table. You will see the table formatted into five columns:
Name: displays the apps name and icon
Icon: select this icon to appear on your Application Dock (limit of 10)
Auto: specify start-up parameters
Skin: Apply a theme to the application's window
Remove: remove applications you have installed

Auto Launcher

In the Auto text box, you can specify parameters for the applications at the time that you log-in to your desktop. For example, if you'd like eyeOS to check for new messages in eyeMessages when you sign on, type the command (check) in the Auto line next to the eyeMessages entry.
If you'd like eyeOS to automatically launch your eyeCalendar app when you sign on, type () on the Auto line next to the eyeCalendar entry.
When you're done your changes, scroll down to the bottom of the window and click the Updated app configuration button.

Installing New Apps

Next to the Manage Apps option, you will see the Install Apps option, which you use to install new eyeOS application tar.gz packages to the system.
If you install applications while you're logged in as the Root user, they will be available to all users of the system. If you install applications as a regular user, they will only be available to you.
If you happen to notice in your eyeApps menu two icons for the same application, this is most likely a result of you installing an application that Root has also installed for the entire system. Click on Install Apps. You will be presented with the Application Installer dialogue window. To install an application, click on the Browse button and navigate to where you have the eyeOS application package .tar.gz file stored. Select it and click the Save or Open button (depending on the OS and web browser you're using). The path and name of the file should then appear in the dialogue box within the Application Installer window. Click on the arrow button directly below the dialogue box. This will upload and install the .tar.gz file to the eyeOS system. If the application is successfully installed, you'll see the words Install Complete appear on the bottom frame of the window. You will also see the new icon in your eyeApps window.

For those applications that you've installed, you can easily remove them from the system by clicking on the cross icon (X) under the Remove column to deleted the app.
