Introduction

eyeHome acts as your account's "home base". From here, you can view and manage the files stored in your personal storage space, view and send messages to and from other system users, create new text documents, add to your contacts list, and install new applications (to your personal account).
The eyeHome window is divided into two parts, the Actions table and the Home directory table.

Action Items

From your Actions table, you have quick access to some of the more frequently used eyeOS apps: New document - eyeEdit, New Message - eyeMessages, New Contact - eyePhones, and Install Application - eyeOptions.
Under Notifications, eyeHome will indicate new messages in your eyeMessages account by changing "No new Messages" to "New Messages". Click on this to open up your eyeMessages folder.

Managing Files

To the right of the Actions table, you will be presented with a list of files in your home directory. If you have no files stored under your home directory, eyeHome will simply display Your home Dir is empty.
To upload new files to your home directory, specify the name of the file you wish to upload from your PC's hard drive in the Upload a file box. Use the Browse... button to navigate to the file stored on your hard drive.
Click on the Upload button (oval button with the arrow pointing to the right) directly below the file name box to tranfer the file to your eyeOS home directory.
Files that are stored in your home folder will be listed in the following way:
File icon: an icon representing the file type.
File name: the name of the file (with extension).
File size: the size of the file.
File date: the date and time the file was uploaded.
View icon: if the file is a type that can be viewed by eyeViewer, a view icon (in the form of a bulls-eye) will be shown. Click on it to view the document. eyeViewer can display .html, .txt, .jpg, .png, and .bmp files
Download icon: click on this icon to download the file from your home directory to the PC's hard drive.
Delete icon: click on this icon to delete the file from your home directory, moving to the eyeTrash folder. You can view the contents of eyeTrash by clicking on the Recycle (or Trash) icon, located in the bottom right-hand corner of your eyeOS desktop.
Note: The size of the files that are acceptable for uploading to your eyeOS home directory will be determined by the system settings in the PHP config files. Any files exceeding the set limits will be rejected.
