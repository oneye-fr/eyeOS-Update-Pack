Einleitung
EyeHome ist für die Verwaltung Ihrer Daten im eyeOS-System zuständig. Sie starten eyeHome, indem Sie auf die beiden grünen Kreise auf Ihrem eyeOS-Desktop klicken.
Das eyeHome-Fenster zeigt Ihnen eine Übersicht Ihres Home-Ordners und bietet zudem mehrere Möglichkeiten Termine usw. zu verwalten.
Von eyeHome aus können Sie die wichigsten Programme ausfüren und Ihre Dateien verwalten.

Aktionen
Einen neuen Ordner erstellen Sie über diesen Button: Geben Sie den gewünschten Ordnernamen an und klicken Sie auf den kleinen grünen Pfeil.
Sie laden eine Datei von Ihrem PC ins Internet, indem Sie diesen Button betätigen: Beachten Sie jedoch, dass Sie auf den meisten Servern lediglich Dateien mit einer Größe von 2 MB hochladen können. Als Administrator können Sie dies jedoch mit einer php.ini beheben.

Außerdem zeigt eyeHome an, ob und wieviele neue Nachrichten in eyeMessages eingetroffen sind.

Dateiverwaltung
In eyeHome gibt es zwei verschiedene Bereiche, in denen Sie Ihre Dateien speichern können. Es gibt Ihre persönlichen Dateien und die öffentlichen Dateien. Die öffentlichen Dateien kann jeder einsehen. Veröffentlichen Sie eine Datei, so wird sie hier angezeigt. Lediglich Sie können die Datei löschen. Jedoch alle anderen eyeOS-Benutzer können sich die Datei in ihre persönlichen Dateien kopieren.
Die persönlichen Dateien sind nur für Sie bestimmt und von keinem anderen Benutzer einsehbar.

Die Ordnerstruktur wir am oberen Rand angezeigt.

Die Dateiansicht:

Ordner
Datei (*.*)
Bild (*.bmp,*.gif,*.jpeg,*.jpg,*.png,*.svg,*.tiff)
Text (*.htm,*.html,*.txt)
Word-Dokument (*.doc,*.odt,*.sxw,*.sdw)
Excel-Dokument (*.xls,*.ods,*.sxc,*.sdc)
PowerPoint-Dokument (*.ppt,*.odp,*.sxi,*.sdd)

Links sehen Sie eines von vier Bildern, dass den Dateitypen zugeordnet ist. Daneben finden Sie den Dateinamen, über den Sie Ordner öffnen und Text- oder Bilddateien im eyeViewer ansehen können. Bei einer nicht definierten Datei startet ein Download.

Klicken Sie auf den Einstellungen-Button erscheint ein Menü, über das Sie Die Datei downloaden, löschen oder veröffentlichen bzw. in die persönlichen Dateien kopieren können. Beim Löschen werden die Dateien jedoch nur in den Papierkorb verschoben, von wo aus Sie diese dann endgültig löschen oder auch wiederherstellen können.
Ordner können nur gelöscht werden, müssen dafür leer sein und werden nicht in den Papierkorb verschoben
