Einleitung
EyeEdit ist eine professionelle Textverarbeitung. Sie basiert auf TinyMCE, ein weit verbreiteter Text-Editor im Internet, und bietet viele und auch sehr ausgereifte Funktionen.
In den Einstellungen können Sie den RTF(=Rich-Text-Format)-Editor aktivieren und deaktivieren. Wenn der RTF-Editor aktiviert ist, können Sie Texte formatieren. Wenn er jedoch deaktiviert ist, können Sie nur einfache Texte schreiben. Der Standard ist mit aktiviertem RTF-Editor.
Achtung: Nach dem Ändern der Einstellung müssen Sie eyeEdit neu starten.

Dokumente verwalten
Nach dem Start wird automatisch immer ein neues Dokument geöffnet.
neues Dokument Klicken Sie auf diesen Button, um ein leeres Dokument zu laden. Nicht gespeichertes geht dabei verloren.
Dokument öffnen Um ein Dokument zu öffnen, verwenden Sie diesen Button. Klicken Sie dann auf den jeweiligen Dokumentnamen. Um ein Dokument zu löschen klicken Sie auf das rote X. Aber Achtung, eine gelöschte Datei wird nicht in den Papierkorb verschoben, sondern  direkt gelöscht.
Dokument speichern Nach dem Klick auf dieses Icon geben Sie den Titel des Dokuments ein. Aktivieren Sie HTML-Kopie im Home-Verzeichnis speichern, wenn Sie eine HTML-Datei in Ihr Home-Verzeichnis kopiert haben möchten. Diese Datei können Sie dann downloaden oder ansehen. Geben Sie hinter Status an, ob das Dokument von jedem eyeOS-Benutzer geändert und angesehen werden kann (Öffentlich), oder nur von Ihnen (Privat).

Tipp: Wenn Sie eine "*.html"- oder eine "*.htm"-Datei mit eyeEdit bearbeiten möchten, benennen Sie diese offline um in eine "*.eyeNote"-Datei. Erstellen Sie zusätzlich eine Datei mit dem selben Namen plus ".xml". Editieren Sie nun die zweite Datei mit einem Editor und setzen Sie den Inhalt folgendermaßen ("Titel" ersetzen Sie bitte durch einen belibiegen Titel für das Dokument):
<?xml version='1.0'?>
<eyeEdit>
  <title>"Titel"</title>
</eyeEdit>

Bilder & Tabellen
Um ein Bild einzufügen, klicken Sie auf den entsprechenden Button in der Menüleiste und geben Sie hinter Bild-URL die URL (den Pfad) zu Ihrem Bild an. Wenn Sie ein Bild aus Ihrem Home-Verzeichnis einfügen möchten, benutzen Sie diese URL: home/<Benutzer>/ ersetzen Sie <Benutzer> durch Ihren Benutzernamen und fügen Sie den Namen des Bildes an, und gegebenenfalls auch die Unterordner des Home-Verzeichnisses.

Eine Tabelle fügen Sie über den Button, Neue Tabelle einfügen hinzu. Es öffnet sich wieder ein neues Fenster. Hier können Sie alle nötigen Informationen angeben.
