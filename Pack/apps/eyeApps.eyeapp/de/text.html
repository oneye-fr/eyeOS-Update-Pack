Einleitung
EyeApps ist die Programmverwaltung für Ihr eyeOS-System. Sie starten eyeApps, indem Sie auf das Plus auf dem eyeOS-Desktop klicken.
Das eyeApps-Fenster zeigt Ihnen eine Übersicht der momentan installierten Programme. Um ein Programm zu starten klicken Sie einfach auf das jeweilige Programm-Icon.
Von eyeApps aus können Sie Programme verwalten, de- und installieren, Die Desktop-Symbole auswählen und den Autostart konfigurieren.

Programme verwalten
Nach dem Start von eyeApps sehen Sie die installierten Programme tabellarisch aufgelistet. Es gibt fünf Spalten:
Symbol: Symbol des Programms - Ein klick auf es startet das Programm
Name: Name des Programms
Desktop: Zeigt an, ob das Programm auf dem Desktop angezeigt wird - max. 15 Icons
Autostart: Zeigt den Autostart-Status an - mehr dazu hier
Wiederhelstellen: Wiederherstellung der Standard-Einstellungen des jeweiligen Programms
Löschen: Deinstalliert das Programm - mehr dazu hier

Programme de- und installieren
Wollen Sie ein Programm installieren, so müssen Sie sich das Programm zu erst auf www.eyeapps.org downloaden.
Klicken Sie nun, nach dem Start von eyeApps, auf das Icon Programm installieren. Klicken Sie nun auf Durchsuchen... und geben Sie das Programm-Archiv an. Wenn Sie der root-Benutzer sind, können Sie noch wählen, ob das Programm systemweit installiert werden soll. Das bedeutet, dass jeder Benutzer das Programm verwenden kann. Ein letzter Klick auf den grünen Pfeil schließt dann die Installation ab. Bei größeren Programmen müssen Sie gelegentlich einige Sekunden oder Minuten warten. Jedoch sollte eine Installation eines Programms, das gepackt nur wenige KB oder bis 5 MB hat keine 10 Minuten dauern. Sollte das jedoch widererwartend auftreten Beenden Sie Ihren Browser und starten Sie ihn neu.
Wenn Sie sich entschließen ein Programm zu deinstalliern, dann müssen Sie lediglich auf das Löschen-Icon klicken und die erscheinende Meldung mit OK beantworten.
Beacten Sie: Einige Programme lassen sich nie, und andere nur von dem root-Benutzer löschen !

Autostart konfigurieren
Die Autostart-Funktion ist zu vergleichen mit dem Autostart-Ordner im Startmenü unter Windows. Jedes Programm kann beim einloggen gestartet werden. Geben Sie dazu in die Textbox des jeweiligen Programms die gewünschten Parameter ein.

Die wichtigsten Parameter sind hier aufgelistet:

Programm Aktion einzugebender Text
alle Programme Programm starten ()
eyeBoard auf neue Einträge prüfen (check)
eyeCalendar auf Termin prüfen (Einstellung) (check)
eyeCalendar auf Termin prüfen (manuelle Anzahl) (check,1)
eyeMessages auf neue Nachrichten prüfen (check)
eyeNav Adresse öffnen (z.B.: www.eyeos.org) (http://www.eyeos.org/)
