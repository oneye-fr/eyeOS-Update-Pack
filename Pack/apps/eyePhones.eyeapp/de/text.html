Einleitung
EyePhones ist Ihre persöhnliche Kontaktverwaltung.
Nach dem Start sehen Sie eine Übersicht aller Ihrer Kontakte:
Nachname, Vorname Telefon Bearbeiten Löschen
Mustermann, Max 01234 / 56789

Mit einem Klick auf das Bearbeiten-Symbol oder auf den Namen können Sie sich alle Informationen ansehen und ändern. Ein Klick auf den Löschen-Button löscht den Kontakt aus eyePhones.

Kontakte verwalten
Legen Sie einen neuen Kontakt über das Icon Neuer Kontakt an. Füllen Sie alle von Ihnen gewünschten Textfelder aus und klicken Sie auf speichern. Sie gelangen wieder zur Startseite.
Sie löschen einen Kontakt über den Löschen-Button. Über das Bearbeiten-Symbol oder mit Klick auf den Namen können Sie alle Daten wieder ändern. Vergessen Sie aber nicht auf den Speichern-Button zu klicken.
