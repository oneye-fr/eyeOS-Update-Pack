installation
------------

To install eyeOS you have to do this:
Um eyeOS zu installieren gehen Sie folgendermaßen vor:

 1. Download the latest version of eyeOS from http://www.eyeOS.org.

 2. Uncompress the package.
 3. Upload the files and directories to you Webspace.
 4. Set the CHMOD of the directorie "etc" to 777.
 5. Start eyeOS in your browser and install it (only 1 step).

The installation is complete. You can start eyeOS.

problems:
 - Your browser is only white:
    - Set your webspace to PHP 4 or if you can set it up to PHP 5.
 - You read, that files or directories cann`t be  written:
    - Create a php.ini  and set its text to "Safe_mode = Off".
    - Test the point 4.
