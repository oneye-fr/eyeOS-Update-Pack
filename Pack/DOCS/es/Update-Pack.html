Update-Pack
-----------

To install the Update-Pack, do this:

 1. Install eyeOS.
     (If you want to update a eyeos system, leave this step out!
 2. Download the newest version of the Upadte-Pack from www.eyeos.de.md.
 3. Extract the file.
 4. Now copy the files into your eyeos system.
    Replace all old files and folders with its new.

The Update-Pack has been installed now.
